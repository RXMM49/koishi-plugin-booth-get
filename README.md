# koishi-plugin-booth-get

- [NPM Package](https://www.npmjs.com/package/koishi-plugin-booth-get)
- [GitHub Repository](https://github.com/Unbloomed-flowers/koishi-plugin-booth-get)

## 介绍

简单获取 booth.pm 页面的插件。

## 功能介绍

本插件适用于获取以下类型内容：
- VRChat 相关商品
- MMD 模型
- 周边商品
- 游戏
- Live2D 资源
- 视频内容

## 使用方法

### 基础命令

1. **获取指定商品信息**
摊位 <商品ID>


2. **搜索商品**
摊位名称 <关键词> [-a <作者>]

示例：
摊位名称 模型 摊位名称 模型 -a 作者名


3. **查看作者店铺**
摊位作者 <作者名>


### 订阅功能

1. **订阅作者**
摊位订阅 <作者名或链接>


2. **取消订阅**
摊位退订 <作者名或链接>


3. **查看订阅列表**
摊位订阅列表


### 自动解析

插件会自动识别消息中的 booth.pm 链接并生成商品或店铺卡片。

### QQ Markdown Button 支持

插件支持使用 QQ Markdown Button 格式发送消息（实验性功能）：
1. 安装 `koishi-plugin-qq-markdown-button` 插件
2. 在插件配置中启用 "启用QQ Markdown Button支持"
3. 插件将会尝试使用 Markdown 格式发送消息（如果可用）

## 特点说明

- 在使用 VRChat 相关搜索时，由于 booth.pm 的特殊性，《摊位名称》命令会以最新发布商品为第一检索结果。
- 默认情况下，插件会自动获取最新发布商品，如需获取其他商品，请自行修改插件代码。
- 后续更新会以卡片样式 web 版进行更新，优化更好且简介的样式面板。

## 配置选项

- **加载超时时间**：设置页面加载的最长时间，默认为 10 秒
- **空闲超时时间**：等待页面空闲的最长时间，默认为 30 秒
- **代理服务器**：可配置代理服务器地址
- **R18 内容检测**：启用后会过滤包含 R18 标签的内容
- **更新检测间隔**：检测订阅作者更新的时间间隔（分钟），默认为 30 分钟
- **启用QQ Markdown Button支持**：启用后会尝试使用QQ Markdown格式发送消息（需要安装koishi-plugin-qq-markdown-button插件）
